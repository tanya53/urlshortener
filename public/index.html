API Basejump : URL Shortner

User stories:

  • I can pass a URL as a parameter and recieve a shortened URL in a JSON response.
  • When I visit that shortened URL, I will be redirected to the original link

Example usage:

https://urlshortener-tanya53.c9users.io/new/https://www.google.com

https://urlshortener-tanya53.c9users.io/new/http://freecodecamp.com/news

Example creation output: (url to shortened)

{ "original_url": "http://freecodecamp.com/news", "short_url": "https://urlshortener-tanya53.c9users.io/4" }

Example: shortened redirect

https://urlshortener-tanya53.c9users.io/4

redirect to:

http://freecodecamp.com/news
